    广度优先搜索
      • 简介
      • 基本原理
      • 节点层数

    简介 $

    广度优先搜索（Breadth First Search，简称 BFS）按照层次顺序查找节点，如下图：

    1. 访问节点 0（根节点）；

    2. 访问节点 0 的所有子结点（结点 1、2）；

    3. 访问节点 1、2 的所有子结点（结点 3、4、5）；

      $\cdots \ \cdots$

    重复访问子结点的操作，直到访问到目标节点。

    基本原理 $

    1. 定义一个队列，用于存放节点编号；

    2. 将根节点编号插入队尾（入队）；

    3. 将队头的所有子节点入队，移除队头元素（出队），重复直到队列为空。

    Warning

    为了避免重复访问同一节点，已经进过队列的结点不能再次入队。

    bool vis[]; // vis[i] 代表节点 i 是否进过队列

void bfs(int s, int e) { // 在根节点为 s 的图中找到节点 e
memset(vis, false, sizeof vis);
queue<int> Q; Q.push(s); // 根节点入队
while(!Q.empty()) {
    int top = Q.top(); Q.pop(); // 出队
        if(top == e) return; // 找到结点 e
        for(int i = top 的所有子节点编号)
            if(!vis[i]) // 如果没有进过队列
                Q.push(i), vis[i] = true; // 入队
    }
}

    节点层数 $

    求任意节点所在的层数（深度）。

    • 子节点层数 $=$ 父节点层数 $+$ $1$。
    struct Node {
    int id, step; // id 为节点编号，step 为层数
};

bool vis[];

int bfs(int s, int e) { // 在根节点为 s 的图中，返回节点 e 的层数
    memset(vis, false, sizeof vis);
    queue<Node> Q; Q.push(Node{s, 1});
    while(!Q.empty()) {
        Node top = Q.top(); Q.pop();
        if(top.id == e) return top.step;
        for(int i = top 的所有子节点编号)
            if(!vis[i]) {
                Q.push(Node{i, top.step + 1}); // 子节点层数 = 父节点层数 + 1
                vis[i] = true;
            }
    }
}
