    重要性质 $

    • $a∧a=0$

    • $a∧0=a$

    • $a\; \& \; 0=0$

    • $a\; \& \; -1=a$

      注：$-1$ 的补码用二进制表示为 $11111111$（每一位都为 $1$）。

    min/max(a, b) $

    通常我们会这样定义 $min(a, b)$：

    int min(int a, int b) { // if 语句
    if(a < b) return a;
    else return b;
}

int min(int a, int b) { // 三目运算符
    return a < b ? a : b;
}

    但在某些机器上，执行「$if()$ 语句」或「三目运算符」很浪费时间，于是有人想出了更快的版本：

    int min(int a, int b) {
    return b ^ ((a ^ b) & - (a < b));
}

    乍一看还挺神奇。现在我们逐步理解该语句。首先看最右边的 $-(a<b)$：

    $$ -(a < b) =\left\{\begin{aligned} &-1, &&a < b\\ &0, &&a≥b \end{aligned}\right. $$

    根据 重要性质，有：

    • $a<b$ 时，$(a∧b)\&\color{blue}{-(a<b)}\color{black}=(a∧b)\&\color{blue}{-1}\color{black}=a∧b$；

    • $a≥b$ 时，$(a∧b)\&\color{blue}{-(a<b)}\color{black}=(a∧b)\&\color{blue}{0}\color{black}=0$。

    于是继续推下去：

    • $a<b$ 时，$b∧\color{blue}{((a∧b)\&-(a<b))}\color{black}=b∧\color{blue}{(a∧b)}\color{black}=a$；

    • $a≥b$ 时，$b∧\color{blue}{((a∧b)\&-(a<b))}\color{black}=b∧\color{blue}{0}\color{black}=b$。

    因此 $min(a, b)=b∧((a∧b)\&-(a<b))$。

    同理，也能推出 $max(a, b)$：

    int max(int a, int b) {
    return a ^ ((a ^ b) & - (a < b));
}

    swap(a, b) $

    交换 $a,b$ 的值，一般需要用到「第三个变量」：

    void swap(int& a, int& b) {
    int c; // 第三个变量
    c = a, a = b, b = c;
}

    利用二进制原理，可以省去「第三个变量」占用的内存：

    void swap(int& a, int& b) {
    a ^= b ^= a ^= b;
    // 这条语句等价于 a ^= b, b ^= a, a ^= b;
}

    举个例子就能秒懂：

    $a$ $b$
    初始值 $1$ $2$
    $a∧=b$ $1∧2$ $2$
    $b∧=a$ $1∧2$ $2∧1∧2$
    $a∧=b$ $1∧2∧2∧1∧2$ $2∧1∧2$
    计算结果 $2$ $1$

    确定 x 是否为 2 的幂 $

    判断一个数能否由若干个 $2$ 相乘得到。初学者一般都这么写：

    bool judge(int x) {
    int a = 1;
    while(a <= x) {
        if(a == x) return true;
        a *= 2;
    }
    return false;
}

    时间复杂度为 $O(\log{n})$。也许有人对此不是很满意。

    于是互联网上的高手给出了数个 $O(1)$ 的解法：

    bool judge(int x) {
    return x && !(x & (x - 1));
}

bool judge(int x) {
    return !((1 << 31) % x);
}

bool judge(int x) { // 用到了 lowbit() 的知识
    return x == (x & -x);
}

    都是很浅显易懂的写法，然而没有多少人能在第一时间想到 $\cdots \ \cdots$

    确定比 x 大的最小 2 的幂 $

    比 $3$ 大的最小「$2$ 的幂」是 $4=2^2$，比 $62$ 大的最小「$2$ 的幂」是 $64=2^6$。依此类推。

    $O(\log{n})$ 做法：

    int nextPow(int x) {
    int a = 1;
    while(a < x) {
        a *= 2;
    }
    return a;
}

    $O(1)$ 做法：

    int nextPow(int x) {
    x --;
    x |= x >> 1;
    x |= x >> 2;
    x |= x >> 4;
    x |= x >> 8;
    x |= x >> 16;
    x ++;
    return x;
}
