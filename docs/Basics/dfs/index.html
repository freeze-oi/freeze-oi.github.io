    深度优先搜索（Depth First Search，简称 DFS）在查找节点时，尽可能往更深处搜索，如下图：

    基本原理 $

    DFS 算法在本质上是一种递归：

    • 若当前结点为目标结点，直接返回；

    • 递归搜索当前结点的每个子节点。

    void dfs(int s, int e) { // 从节点 i 出发，查找结点 e
    if(s == e) return; // 找到结点 e
    for(int i = 结点 k 的子节点编号)
        dfs(i, e); // 搜索
    return; // 回溯（此处可以省略）
}
