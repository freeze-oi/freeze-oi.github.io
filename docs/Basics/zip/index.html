      • Lowbit 运算

    简介 $

    将长度为 $m$ 的 $bool$ 数组用一个 $m$ 位二进制整数表示，该整数的第 $i$ 位对应原数组第 $i$ 个元素。

    操作 运算
    取出二进制数 $n$ 的第 $k$ 位 (n >> (k - 1)) & 1
    将二进制数 $n$ 的第 $k$ 位取反 n ^= (1 << (k - 1))
    将二进制数 $n$ 的第 $k$ 位赋值为 $1$ n |= (1 << (k - 1))
    将二进制数 $n$ 的第 $k$ 位赋值为 $0$ n &= (~(1 << (k - 1)))

    Lowbit 运算 $

    $lowbit(n)$ 为 $n$ 在二进制表示下「最低位的 $1$ 和其后所有的 $0$」构成的数。例如 $n=(101000)_2$，$lowbit(n)=(1000)_2$。

    $n[i]$：$n$ 在二进制表示下的第 $i$ 位数字。假设 $n[k]=1$，$n[0\cdots k-1]=0$。

    1. 将 $n$ 的每一位取反，即 $n=\sim n$，此时 $n[k]=0$，$n[0\cdots k-1]=1$，其它位和原来相反；

    2. 令 $n=n+1$，此时 $n[k]=1$，$n[0\cdots k-1]=0$，其它位仍和原来相反；

    3. 因此 $n\&(\sim n+1)$ 仅有第 $k$ 位为 $1$。

    由于在补码表示下 $-n=\sim n+1$，因此：

    $$ lowbit(n)=n\&(\sim n+1)=n\&-n $$

    下面给出几个样例，辅助理解：

    $n$ $\sim n$ $\sim n+1$ $n\&(\sim n+1)$
    $(101000)_2$ $(010111)_2$ $(011000)_2$ $(001000)_2$
    $(101100)_2$ $(010011)_2$ $(010100)_2$ $(000100)_2$
