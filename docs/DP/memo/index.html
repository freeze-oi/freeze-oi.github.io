    记忆化搜索 $

    考虑开数组 $F[ \ ]$ 保存计算结果，从而避免重复计算。

    1. 若 $f(x)$ 未被调用，算出 $f(x)$ 的结果，并存入 $F[\text{x}]$；

    2. 若 $f(x)$ 已经被调用过，直接返回 $F[\text{x}]$。

    和 动态规划 中的填表思想如出一辙。

    int F[] = {0, 1, 1};
int f(int x) {
    if(F[x] != 0) return f[x]; // 如果 F[x] ≠ 0，则 f(x) 已经被调用过
    return F[x] = f(x - 1) + f(x - 2); // 返回时保存
}
