    简介 $

    递归访问树中所有节点的过程叫做树的遍历。本章介绍二叉树的 $3$ 种遍历方式。

    前序遍历 $

    先访问根节点，再遍历左子树，最后遍历右子树。上图的前序遍历顺序为 $1→2→4→5→3$。

    void dfs(int u) { // 打印以 u 为根的树的遍历顺序
    if(!u) return;
    cout << u << endl;
    dfs(l[u]); // l[u] 表示 u 的左子节点
    dfs(r[u]); // r[u] 表示 u 的右子节点
}

    中序遍历 $

    先遍历左子树，再访问根节点，最后遍历右子树。上图的中序遍历顺序为 $4→2→5→1→3$。

    void dfs(int u) {
    if(!u) return;
    dfs(l[u]);
    cout << u << endl;
    dfs(r[u]);
}

    后序遍历 $

    先遍历左子树，再遍历右子树，最后访问根节点。上图的后序遍历顺序为 $4→5→2→3→1$。

    void dfs(int u) {
    if(!u) return;
    dfs(l[u]);
    dfs(r[u]);
    cout << u << endl;
}
