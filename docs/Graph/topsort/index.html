    拓扑排序
      • AOV 网

    AOV 网 $

    给出几组关系：$a > b$，$a > c$，$b > c$，$b > d$，$\cdots$，求可能的 $a \ b \ c \ d \cdots$ 的大小顺序？

    将上述关系转化为有向图，其中 $a → b$ 代表 $a > b$。这类能表示节点大小关系的图称做 AOV 网。

    基本原理 $

    拓扑排序 求的是合法的优先顺序，求得的序列称为 拓扑序列。同一张 AOV 网可能有多个合法的拓扑序列。

    Info

    没被箭头指着的节点（入度为 $0$ 的节点）是当前最大的节点。

    节点 $a$ 的入度为 $0$。在拓扑序列中追加 $a$，并删除 $a$ 和它的所有邻边：

    拓扑序列：$a$。

    节点 $e$ 的入度为 $0$。在拓扑序列中追加 $e$，并删除 $e$ 和它的所有邻边：

    拓扑序列：$a \ e$。

    节点 $b$ 的入度为 $0$。在拓扑序列中追加 $b$，并删除 $b$ 和它的所有邻边：

    拓扑序列：$a \ e \ b$。

    故样例的一种拓扑序列为 $a \ e \ b \ c \ d \ f$。

    $deg[i]$ 表示节点 $i$ 的入度，在输入时可以预处理。

    1. 定义一个队列，用于存放结点；

    2. 将所有入度为 $0$ 的节点入队；

    3. 将队头结点出队，并删除该结点的所有出边。若出现新的结点入度为零，则将其入队。

    拓扑排序结束时，若拓扑序列未包含全部节点，则说明剩下的节点形成了闭环。此时不存在合法拓扑序列。

    vector<int> topSort() {
    vector<int> vec;
    queue<int> q;
    for(int i = 1; i <= n; i ++)
        if(!deg[i]) q.push(i);
    while(!q.empty()) {
        int top = q.front(); q.pop();
        vec.push_back(top);
        for(int i = 0; i < g[top].size(); i ++)
            if(! -- deg[g[top][i]])
                q.push(g[top][i]);
    }
    if(vec.size() != n) return vector<int>();
    return vec;
}
