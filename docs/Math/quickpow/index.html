    快速幂 $

    如何快速求出 $a^b$（$a,b\in\mathbb{Z}$）？

    设 $Pow(a, b)$ 表示 $a^b$。

    • 若 $b$ 为偶数，则 $a^b=a^{b/2}×a^{b/2}$，即 $Pow(a, b)=Pow(a, b/2)×Pow(a, b/2)$。

    • 若 $b$ 为奇数，则 $a^b=a^{b/2}×a^{b/2}×a$（向下取整），即 $Pow(a, b)=Pow(a, b/2)×Pow(a, b/2)×a$。

    边界条件：$Pow(a, 0)=1$。时间复杂度为 $O(\log{b})$。

    typedef long long LL;

LL Pow(LL a, LL b) { // 计算 a^b
    if(!b) return 1; // 边界条件
    LL res = Pow(a, b / 2);
    if(b % 2 == 0) // b 是偶数
        return res * res;
    else // b 是奇数
        return res * res * a;
}

    事实上，快速幂还可以写得更清爽：

    typedef long long LL;

LL Pow(LL a, LL b) {
    if(!b) return 1;
    LL res = Pow(a, b / 2);
    return res * res * (b % 2 ? a : 1);
}

    递推版快速幂（一次性求出 $a^0\sim a^{N}$）：

    typedef long long LL;
const LL N = 1e6;
LL a, Pow[N];

Pow[0] = 1; // 边界条件，a^0 = 1

for(int i = 1; i < N; i ++)
    Pow[i] = Pow[i / 2] * Pow[i / 2] * (i % 2 ? a : 1);
