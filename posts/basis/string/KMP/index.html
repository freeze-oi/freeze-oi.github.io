    KMP 算法

    简介$

    \(KMP\) 算法并不能用来看毛片，它是一种字符串匹配算法。\(KMP\) 是该算法的发明者 \(Kruth\)，\(Morris\) 和 \(Pratt\) 的名字缩写。

    基本原理$

    引出问题$

    给定字符串 \(A\)（长度为 \(m\)）和 \(B\)（长度为 \(n\)），问 \(A\) 中是否包含 \(B\) ？

    首先，比较 \(A\) 和 \(B\) 的第一个字符。因为 \(b\) 与 \(a\) 不匹配，所以 \(B\) 向后移一位。

    继续比较，直到不匹配时，再将 \(B\) 向后移。

    这么做虽然可行，但是效率太低。

    暴力算法
    bool involve(string a, string b) {
    int m = a.size(), n = b.size();
    int i = 0, j = 0;
    while(i < m) {
        while(j < n && a[i] == b[j]) j ++;
        if(j == n) return true;
        i ++, j = 0;
    }
    return false;
}
    暴力算法（\(\textcolor{red}{×}\)） \(KMP\) 算法（\(\textcolor{green}{√}\)）
    \(\textcolor{red}{O(mn)}\) \(\textcolor{green}{O(m+n)}\)

    预处理$

    其实，刚才匹配到这一步时，\(B\) 串可以直接向后移 \(2\) 位，从黄色处开始比较。

    为什么呢？因为 \(B\) 串的这两段相等。

    而此时 \(A\) 串和 \(B\) 串绿色部分必定相等，黑框框住的这两段也相等。

    因此 \(A\) 串和 \(B\) 串的这两段也相等，可以直接把 \(B\) 串往右移，直到这两段处于同一竖直位置。

    \(KMP\) 算法快就快在这。

    前缀&后缀$

    字符串 \(abcde\) 的「前缀」有：

    \(\begin{align} &a\\ &ab\\ &abc\\ &abcd\\ \end{align}\)

    它的「后缀」有：

    \(\begin{align} e&\\ de&\\ cde&\\ bcde&\\ \end{align}\)

    并且前后缀长度 \(<\) 字符串本身的长度。

    若字符串的某个前缀和某个后缀相等，则相等的这部分称为「公共前后缀」。例如 \(abcabc\) 的公共前后缀为 \(abc\)。

    next 数组$

    \(next[i]\)：\(B[0\cdots i]\) 中最长的「公共前后缀」的长度。注意字符串的下标从 \(0\) 开始。

    如下图，单独取出 \(B\) 串的前 \(4\) 个字符（\(B[0\cdots 3]\)），发现 \(B[0\cdots 1]=B[2\cdots 3]\)，且它们的长度为 \(2\)，因此 \(next[3]=2\)。

    公共前后缀也可以重叠。下图中 \(next[4]=3\)。

    以字符串 \(abacabac\) 为例：

    \[ \begin{align} &a\textcolor{lightgrey}{bacabac}&next[0]=0\\ &ab\textcolor{lightgrey}{acabac}&next[1]=0\\ &\underline{\textcolor{red}{a}}b\underline{\textcolor{red}{a}}\textcolor{lightgrey}{cabac}&next[2]=1\\ &abac\textcolor{lightgrey}{abac}&next[3]=0\\ &\underline{\textcolor{red}{a}}bac\underline{\textcolor{red}{a}}\textcolor{lightgrey}{bac}&next[4]=1\\ &\underline{\textcolor{red}{ab}}ac\underline{\textcolor{red}{ab}}\textcolor{lightgrey}{ac}&next[5]=2\\ &\underline{\textcolor{red}{aba}}c\underline{\textcolor{red}{aba}}\textcolor{lightgrey}{c}&next[6]=3\\ &abacabac&next[7]=0\\ \end{align} \]

    模板$

    代码
    int next[];

void getnext(string str) {
    next[0] = -1;
    int i = 1, j = 0;
    while(i < str.size()) {
        while(j > -1 && str[i] != str[j]) j = next[j];
        next[++ i] = ++ j;
    }
}

void kmp(string str, string substr) {
    getnext(substr);
    int i = 0, j = 0;
    while(i < str.size()) {
        while(j > -1 && str[i] != substr[j]) j = next[j];
        if(j == substr.size() - 1)
            printf("Fount pattern at %d\n",i - j), j = next[j] - 1;
        i ++, j ++;
    }
}
