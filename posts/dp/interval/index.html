    区间 DP

    简介$

    区间 DP 以「区间长度」作为 DP 的「阶段」，通过小区间的最优值推出大区间的最优值。

    基本原理$

    Quote

    Freeze 懒得写基本原理，他认为这些例题已经代表了区间 DP 的精髓。

    例题$

    石子合并$

    Question

    题目描述

    \(n\) 堆石子排成一列，第 \(i\) 堆石子重量为 \(A_i\)。每次合并相邻两堆石子为一堆，消耗的体力值为这两堆石子的重量和。求将所有石子合并为一堆，最少消耗多少体力。

    样例输入

    4
4 5 9 4

    样例输出

    43

    数据范围

    \(1≤n≤100\)，所有数据皆在 \(int\) 范围内。

    \(f[l,r]\) 表示合并区间 \([l,r]\) 内的石子（第 \(i\) 堆至第 \(j\) 堆石子）花费的最少体力值。

    枚举 \(k=l→r-1\)：

    • 合并区间 \([l,k]\) 内的石子为一堆石子，消耗体力值 \(f[l,k]\)；

    • 合并区间 \([k+1,r]\) 内的石子为一堆石子，消耗体力值 \(f[k+1,r]\)；

    • 合并剩下两堆石子，消耗体力值 \(A_l+A_{l+1}+\cdots+A_r\)。

    \[ f[l,r]=\min_{l≤k<r}\{f[l,k]+f[k+1,r]\}+\sum_{i=l}^r A_i \]

    使用前缀和进一步优化查询区间石子质量和的效率：

    \[ f[l,r]=\min_{l≤k<r}\{f[l,k]+f[k+1,r]\}+sum[r]-sum[l-1] \]

    由于大区间的「最少体力值」由小区间推出，故计算顺序为从小区间到大区间。

    枚举区间长度 \(len=2→n\)，每次计算所有长度为 \(len\) 的区间的「最少体力值」。

    初始条件 \(f[i,i]=0\)，其余为 \(∞\) 计算顺序 \(f[\)小区间\(]→f[\)大区间\(]\)
    最终答案 \(f[1,n]\) 时间复杂度 \(O(n^3)\)
    Code
    memset(f, 0x7f, sizeof f);
for(int i = 1; i <= n; i ++) {
    f[i][i] = 0;
    sum[i] = sum[i - 1] + a[i];
}
for(int len = 2; len <= n; len ++) // 区间长度
    for(int l = 1; l + len - 1 <= n; l ++) { // 枚举区间左端点
        int r = l + len - 1; // 区间右端点
        for(int k = l; k < r; k ++)
            f[l][r] = min(f[l][r], f[l][k] + f[k + 1][r]);
        f[l][r] += sum[r] - sum[l - 1];
    }

    删数问题$

    Question

    题目描述

    数列 \(A\) 的 \(n\) 个元素互不相同。要求重复从 \(A\) 的左边或右边删掉连续 \(i\) 个数（\(1≤i≤n\)），若剩余不足 \(i\) 个数则全部删除，直到所有数都被删除。

    每次操作都能获得价值。删除 \(A_i\cdots A_k\) 的价值为 \(|A_i-A_k|·(k-i+1)\)，如果只能删一个数，获得的价值为这个数的值。求操作的最大价值。

    样例输入

    6
54 29 196 21 133 118

    样例输出

    768

    数据范围

    \(3≤n≤100,1≤A_i≤1000\)。

    \(f[l,r]\) 表示删除 \(A[l\cdots r]\) 得到的最大价值。删除 \(A[l\cdots r]\) 有两种方法：

    • 一次性删除 \(A[l\cdots r]\)，得到价值 \(|A_l-A_r|·(r-l+1)\)；

    • 先删除 \(A[l\cdots k]\)，再删除 \(A[k+1\cdots r]\)，得到价值 \(f[l,k]+f[k+1,r]\)。

    \[ f[l,r]=\max\left\{\begin{aligned} &|A_l-A_r|·(r-l+1)\\ &f[l,k]+f[k+1,r],l≤k<r \end{aligned}\right. \]
    初始条件 \(f[i,i]=A_i\) 计算顺序 \(f[\)小区间\(]→f[\)大区间\(]\)
    最终答案 \(f[1,n]\) 时间复杂度 \(O(n^3)\)
    Code
    for(int i = 1; i <= n; i ++) f[i][i] = a[i];
for(int len = 2; len <= n; len ++)
    for(int l = 1; l + len - 1 <= n; l ++) {
        int r = l + len - 1;
        f[i][j] = abs(a[l] - a[r]) * (r - l + 1);
        for(int k = l; k < r; k ++)
            f[i][j] = max(f[i][j], f[i][k] + f[k + 1][j]);
    }
