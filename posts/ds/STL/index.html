    STL

    简介$

    STL（Standard Template Library 标准模板库）是一套功能强大的 C++ 模板类和函数，这些模板类和函数可以实现多种流行和常用的算法和数据结构。

    C++ 标准模板库的核心包括以下三个组件：

    组件 描述
    容器 Containers 容器用于储存某一类对象。
    算法 Algorithms 算法作用于容器。STL 提供了各种操作方式，包括初始化、排序、搜索和转换等。
    迭代器 Iterators 迭代器用于遍历容器中的元素。

    对于每个组件，按照以下分组展开介绍：

    分组 内容 分组 内容
    Define 头文件、定义、初始化 Capacity 内存操作
    Access 元素访问 Modifiers 元素增改
    Operations 优化功能 Code 示例程序

    Warning

    使用 STL 需要默认使用命名空间。

    using namespace std;

    容器（Containers）$

    容器的功能较数组更加强大，且 STL 提供了容器的配套算法。

    Vector$

    Vector 容器相当于可调整大小的数组。

    • 支持从末端插入、删除元素；

    • 支持访问、更改任意位置的元素；

    • 支持在任意位置插入、删除元素（效率较慢）。
    Define
    #include <vector> 头文件
    vector<type> vec 创建空容器 vec，用于存放 type 类元素
    vector<type> vec(n) 创建包含 n 个 type 类元素的容器 vec（元素的默认值为 0）
    vector<type> vec(n, val) 创建包含 n 个 type 类元素的容器 vec，元素值为 val
    Capacity
    size() 返回元素个数
    resize(n) 调整大小，使容器包含 n 个元素
    resize(n, val) 调整大小，使容器包含 n 个元素，新元素初始化为 val
    clear() 清空容器
    empty() 返回是否为空
    Access
    operator [pos] 返回第 pos 个元素的引用（pos 从 0 记起）
    at(pos) 返回第 pos 个元素的值
    front() 返回前端元素的值
    back() 返回末端元素的值
    Modifiers
    assign(n, val) 清空容器，并添加 n 个 val
    push_back(x) 在末端插入 x
    pop_back() 删除末端元素
    insert(pos, x) 在 pos 位置插入 x
    insert(pos, n, x) 在 pos 位置插入 n 个 x
    erase(pos) 删除 pos 位置的元素
    Code
    #include <iostream>
#include <vector>

using namespace std;

int main() {

    vector<int> vec(3, 5); // 创建 int 类的容器，并添加 3 个 5
    /* 当前 vec = {5, 5, 5} */

    for(int i = 0; i < 5; i ++)
        vec.push_back(i); // 在末端插入 i
    /* 当前 vec = {5, 5, 5, 0, 1, 2, 3, 4} */

    vec[2] = 10; // 将第 2 个元素更改为 10（下标从 0 记起）
    /* 当前 vec = {5, 5, 10, 0, 1, 2, 3, 4} */

    cout << vec.size() << endl; // 输出元素个数

    for(int i = 0; i < vec.size(); i ++)
        cout << vec[i] << ' '; // 输出 vec 的第 i 个元素

    return 0;
}

    output:
8
5 5 10 0 1 2 3 4

    Vector<Bool>$

    Vector<Bool> 在 Vector 的基础上，额外支持取反操作（true 变 false，false 变 true）。
    Define
    #include <vector> 头文件
    vector<bool> vec 创建存储 bool 类元素的空容器
    Modifiers
    flip() 全部取反
    flip(pos) 将第 pos 位取反
    Code
    #include <bits/stdc++.h>

using namespace std;

const int arr[] = {0, 1, 0, 0, 1, 0}; 

int main() {

    vector<bool> vec;

    for(int i = 0; i < 6; i ++)
        vec.push_back(arr[i]);

    vec.flip(); // 全部取反 

    for(int i = 0; i < 6; i ++)
        cout << vec[i] << ' ';

    return 0;
}

    output:
1 0 1 1 0 1

    Queue$

    Queue 队列。

    • 支持从末端插入元素，从前端提取元素；

    • 只能访问前端和末端元素。
    Define
    #include <queue> 头文件
    queue<type> que 创建存储 type 类元素的空队列
    Capacity
    size() 返回元素个数
    empty() 返回是否为空
    Access
    back() 返回末端元素的值
    front() 返回前端元素的值
    Modifiers
    push(val) 在末端插入 val
    pop() 删除前端元素
    Code
    #include <iostream>
#include <queue>

using namespace std;

int main() {

    queue<int> que; // 创建 int 类的队列

    for(int i = 0; i < 5; i ++)
        que.push(i); // 在末端插入 i
    /* 当前 que = {0, 1, 2, 3, 4} */

    cout << que.front() << ' ' << que.back() << endl; // 输出前端和末端元素

    que.pop(); // 删除前端元素
    /* 当前 que = {1, 2, 3, 4} */

    cout << que.front(); // 输出前端元素

    return 0;
}

    output:
0 4
1

    Priority Queue$

    Priority Queue 是基于 Queue 的优先队列。

    • 维护前端元素在队列中始终最大（或最小）；

    • 仅支持访问前端元素。
    Define
    #include <queue> 头文件
    priority_queue<type> que 创建存储 type 类元素的优先队列（默认前端最大）
    Capacity
    size() 返回元素个数
    empty() 返回是否为空
    Access
    top() 返回前端元素的值
    Modifiers
    push(val) 在插入 val
    pop() 删除前端元素
    Code
    #include <iostream>
#include <queue>

using namespace std;

const int arr[] = {4, 1, 2, 5, 3, 6}; 

int main() {

    priority_queue<int> que; // 创建优先队列（默认前端最大） 

    priority_queue<int, vector<int>, greater<int> > min_que; // 前端始终最小 

    priority_queue<int, vector<int>, less<int> > max_que; // 前端始终最大

    for(int i = 0; i < 6; i ++) {
        min_que.push(arr[i]);
        max_que.push(arr[i]); // 将 arr[i] 插入队列 
    }

    cout << min_que.top() << endl; // 输出队列最小元素 
    cout << max_que.top() << endl; // 输出队列最大元素 

    min_que.pop(); max_que.pop(); // 删除队列前端元素

    cout << min_que.top() << endl; 
    cout << max_que.top() << endl;

    return 0;
}

    output:
1
6
2
5

    Deque$

    Deque 双向队列相当于 Vector 的升级版。

    • 支持从两端插入、删除元素；

    • 支持访问、更改任意位置的元素（效率较 Vector 稍慢）；

    • 支持在任意位置插入、删除元素（效率较慢）。
    Define
    #include <deque> 头文件
    deque<type> q 创建存储 type 类元素的双向队列
    Capacity
    size() 返回元素个数
    resize(n) 调整大小，使队列包含 n 个元素
    resize(n, val) 调整大小，使队列包含 n 个元素，新元素初始化为 val
    clear() 清空容器
    empty() 返回是否为空
    Access
    operator [pos] 返回第 pos 个元素的引用（pos 从 0 记起）
    at(pos) 返回第 pos 个元素的值
    front() 返回前端元素的值
    back() 返回末端元素的值
    Modifiers
    assign(n, val) 清空容器，并添加 n 个 val
    push_front(x) 在前端插入 x
    push_back(x) 在末端插入 x
    pop_front() 删除前端元素
    pop_back() 删除末端元素
    insert(pos, x) 在 pos 位置插入 x
    insert(pos, n, x) 在 pos 位置插入 n 个 x
    erase(pos) 删除 pos 位置的元素
    Code
    #include <iostream>
#include <deque>

using namespace std;

int main() {

    deque<int> deq(3, 5); // 创建 int 类的双向队列，并添加 3 个 5
    /* 当前 deq = {5, 5, 5} */

    for(int i = 0; i < 5; i ++)
        deq.push_back(i); // 在末端插入 i
    /* 当前 deq = {5, 5, 5, 0, 1, 2, 3, 4} */

    deq[2] = 10; // 将第 2 个元素更改为 10（下标从 0 记起）
    /* 当前 deq = {5, 5, 10, 0, 1, 2, 3, 4} */

    deq.pop_front(); //删除前端元素
    /* 当前 deq = {5, 10, 0, 1, 2, 3, 4} */

    cout << deq.size() << endl; // 输出元素个数

    for(int i = 0; i < deq.size(); i ++)
        cout << deq[i] << ' '; // 输出 deq 的第 i 个元素

    return 0;
}

    output:
7
5 10 0 1 2 3 4

    Stack$

    Stack 栈。

    • 支持从末端插入、提取或访问元素。
    Define
    #include <stack> 头文件
    stack<type> stk 创建存储 type 类元素的空栈
    Capacity
    size() 返回元素个数
    empty() 返回是否为空
    Access
    top() 返回末端元素
    Modifiers
    push(x) 在末端插入 val
    pop() 删除末端元素
    Code
    #include <iostream>
#include <stack>

using namespace std;

int main() {

    stack<int> stk; // 创建 int 类的空栈

    for(int i = 0; i < 5; i ++)
        stk.push(i); // 在末端插入 i
    /* 当前 stk = {0, 1, 2, 3, 4} */

    cout << stk.top() << endl; // 输出末端元素

    stk.pop(); // 删除末端元素
    /* 当前 stk = {0, 1, 2, 3} */

    cout << stk.top(); // 输出末端元素
}

    output:
4
3

    String$

    String 字符串是一种字符容器，相当于 Vector<char> 的升级版。
    Define
    #include <string> 头文件
    string str 创建空字符串
    string str = "context" 创建字符串，并初始化为 context
    Capacity
    size() 返回字符个数
    resize(n) 将字符串长度调整为 n，新元素初始化为空字符
    resize(n, val) 将字符串长度调整为 n ，新元素初始化为 val
    clear() 清空容器
    empty() 返回是否为空
    Access
    operator [pos] 返回第 pos 个字符的引用（pos 从 0 记起）
    at(pos) 返回第 pos 个字符的值
    front() 返回前端字符的值
    back() 返回末端字符的值
    Modifiers
    assign(n, s) 清空字符串，并添加 n 个字符 s
    operator += s 在末端插入字符（或字符串） s
    append(s) 在末端插入字符（或字符串） s
    push_back(s) 在末端插入字符 s
    insert(pos, s) 在 pos 位置前插入字符（或字符串） s
    insert(pos, n, s) 在 pos 位置前插入 n 个字符（或字符串） s
    erase(pos) 删除 pos 位置的字符
    Operations
    c_str() 返回字符串的 char* 指针形式
    substr(pos) 返回从 pos 位置到末尾的子串
    substr(pos, n) 返回从 pos 位置开始，长度为 n 的子串
    Code
    #include <iostream>
#include <string>

using namespace std;

int main() {

    string str = "Do you like"; // 初始化字符串为 "Do you like"

    str += " Van you see?"; // 在末端插入 " Van you see?"
    /* 当前 str = "Do you like Van you see?" */

    str.insert(15, "♂"); // 在位置 15 前插入 "♂"（占 2 字节） 

    cout << str.size() << endl; // 输出字符串长度

    cout << str << endl; // 输出字符串

    cout << str[0]; // 输出位置 0 上的字符 

    return 0;
} 

    output:
26
Do you like Van♂ you see?
D

    Bitset$

    Bitset 支持对二进制整数进行位操作，可当作空间极小的 bool 数组。

    对于二进制整数 \(n\)：

    • 支持访问、更改 \(n\) 任意位的数值（包括取反，赋值）；

    • 支持将 \(n\) 转换为字符串或 unsigned long 形式。
    Define
    #include <bitset> 头文件
    bitset<n> bit 创建 n 位二进制数
    bitset<n> bit(val) 创建 n 位二进制数，并初始化为十进制数 val
    Access
    operator [pos] 返回第 pos 位数的引用（位数从 0 记起）
    test(pos) 返回第 pos 位的值
    size() 返回位数
    count() 返回 1 的个数
    any() 返回是否有 1
    none() 返回是否全部为 0
    Modifiers
    flip() 全部取反
    flip(pos) 将第 pos 位取反
    reset() 全部赋值为 0
    reset(pos) 将第 pos 位赋值为 0
    set() 全部赋值为 1
    set(pos) 将第 pos 位赋值为 1
    set(pos, x) 将第 pos 位赋值为 x
    Operations
    to_string() 返回二进制数的字符串形式
    to_ulong() 返回二进制数的 unsigned long 形式

    Warning

    二进制数的位数从右往左递增，而 bitset 的下标从左往右递增。故数位在 bitset 中倒序储存。

    Code
    #include <iostream>
#include <bitset>

using namespace std;

int main() {

    bitset<3> bit(6); // 创建 3 位二进制数 110（5 转换为二进制是 110）
    /* 当前 bit = {0, 1, 1} = 110 */

    cout << bit.size() << endl; // 输出位数

    bit.flip(0); // 将第 0 位取反
    /* 当前 bit = {1, 1, 1} = 111 */

    bit[2] = 0; // 将第 2 位赋值为 1
    /* 当前 bit = {1, 1, 0} = 011*/ 

    cout << bit.to_string(); // 输出 bit 的字符串形式 

    return 0;
}

    output:
3
011
