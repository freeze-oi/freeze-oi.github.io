    图的存储

    直接存边$

    把每条边的起点、终点、长度存入结构体数组中。

    Code
    struct node {
    int from, to, len;
} edge[];

    也可以分成三个数组来存储。

    Code
    int from[], to[], len[];

    邻接矩阵$

    用二维数组 g[][]（邻接矩阵）存储边长。g[i][j] 表示连接节点 i 和节点 j 的边的长度。

    缺点：不能存储重边；浪费空间。

    邻接表$

    把节点 i 的所有相邻节点插入以 head[i] 开头的链表中（链表最后一个元素的 next 指向 0）。

    Code
    int top, head[];
struct Node {
    int val, len;
    int next; // 这里只需要用到单向链表
} edge[];

void insert(int u, int v, int len) { // 追加一条从 u 到 v 长度为 len 的边
    edge[++ top] = Node{v, len, head[u]};
    head[u] = top;
}

int path(int u, int v) { // 获取以 u, v 为端点的边的长度（没有边则返回 -1）
    for(int p = head[u]; p; p = edge[p].next)
        if(edge[p].val == v) return edge[p].len;
    return -1;
}

    邻接表可以用 STL Vector 实现。

    Code
    struct Node {
    int val, len;
};
vector<Node> edge[];

void insert(int u, int v, int len) {
    edge[u].push_back(Node{v, len});
}

int path(int u, int v) {
    for(int i = 0; i < edge[u].size(); i ++)
        if(edge[u][i].val == v) return edge[u][i].len;
    return -1;
}

    反向边访问$

    使用邻接表存储双向边时，依次存储正向边和反向边，故 edge[i] 和 edge[i + 1]（\(i\) 为偶数）互为反向边。此时 \(i\) 和 \(i+1\) 构成「成对变化」，即 edge[i] 的反向边为 edge[i xor 1]。

    Code
    void insert(int u, int v, int len) { // 追加一条从 u 到 v 长度为 len 的双向边
    edge[++ top] = Node{v, len, head[u]}, head[u] = top; // 从 u 到 v 的边
    edge[++ top] = Node{u, len, head[v]}, head[v] = top; // 从 v 到 u 的边
}

Node rpath(int pos) { // 返回 edge[pos] 的反向边
    return edge[pos xor 1];
}
